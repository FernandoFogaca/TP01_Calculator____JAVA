TP01 – Scientific Calculator Tests

This project was developed for a university assignment at

ESCOLA SUPERIOR DA TECNOLOGIA DA INFORMAÇÃO
Systems Analysis and Development
Back-End Development

The work consists of creating and testing a Scientific Calculator class in Java.

In this project:

Unit 1 – Java project creation and JUnit 5 configuration.
Unit 2 – First unit test implemented and executed.
Unit 3 – Tests written following setup, execution and assertion steps.
Unit 4 – Use of @BeforeEach to create the object before each test.
Unit 5 – Positive scenario test for square root.
Unit 6 – Exception test for negative square root.
Unit 7 – Exception test for division by zero.
Unit 8 – Tests for log and trigonometric functions.
Unit 9 – Code coverage and test importance analysis.
Unit 10 – Review and improvement of test names.
Unit 11 – Organisation of tests inside ScientificCalculatorTest class.

Developed using Eclipse IDE on macOS (MacBook).
